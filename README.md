## pre-commit

Pre-commit verifica i tuoi file prima del commit. Può pulire, formattare
o fare ulteriori controlli su di essi.
Installalo con

        pip3 install pre-commit --user

Eseguilo con

        pre-commit run --all-files


O installalo come pre-commit hook

        pre-commit install
